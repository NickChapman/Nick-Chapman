***Note: The insertion sort runs for a couple of hours***
----------------------------------
		Implementation
----------------------------------
Insertion sort and merge sort are both standard implementations
Radix is implemented using a bucket sort as the digit sorting method

The vectors used for storage are dynamic arrays implemented using Python's
ctypes library.
