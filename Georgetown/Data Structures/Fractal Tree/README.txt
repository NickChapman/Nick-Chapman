----------------------------------
		Running the program
----------------------------------
From the command line type the following:
	python fractal_tree.py

To see configuration parameters for the program run:
	python fractal_tree.py -h

----------------------------------
		Implementation
----------------------------------
The tree is generated using a recursive function which calls upon
the Turtle graphics package. The function itself is quite simple.
